# 工厂车间管理系统 java304

## 项目概述

本项目是基于SpringBoot和Vue的工厂车间管理系统，采用前后端分离的架构设计，适用于多种生产管理场景。

## 技术栈

- **架构：** B/S、MVC
- **系统环境：** Windows/Mac
- **开发环境：**
  - IDEA
  - JDK 1.8
  - Maven
  - MySQL
- **技术栈：**
  - Java
  - MySQL
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

- 管理员
- 用户

### 管理员模块

- 个人中心
- 修改密码
- 个人信息
- 人员管理
- 看板信息管理
- 设备信息管理
- 生产开发管理
- 生产工序管理
- 生产流程管理

### 用户模块

- 个人中心
- 看板信息管理
- 设备信息管理
- 生产开立管理
- 生产工序管理
- 生产流程管理

## 运行环境

- 服务器：推荐使用Java EE兼容服务器如Tomcat
- 数据库：MySQL
- 前端：Vue.js支持的主流浏览器

## 部署步骤

1. 安装并配置好MySQL数据库环境
2. 导入数据库脚本
3. 部署后端服务到Java EE服务器
4. 部署前端代码到Web服务器
